方弦物联：软件全屏 V1.0
===============

> 运行环境要求python3+

## 基础信息

* 模拟器、浏览器等等支持f11全屏的软件的【软件启动、自动置顶、自动全屏】软件

## 打包指令

~~~
pyinstaller .\main.py -F -n 方弦物联：软件全屏-1.0.0 -i .\icon_mini.ico --version-file .\version.ini
~~~
